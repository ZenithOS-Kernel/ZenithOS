# ZenithOS
ZenithOS-Kernel là dự án kernel mã nguồn mở, phát triển từ con số 0 nhằm nghiên cứu kiến trúc hệ điều hành, quản lý phần cứng và xây dựng nền tảng cho ZenithOS trong tương lai.

## Goals
- Learn operating system fundamentals
- Build a kernel from scratch
- Create the foundation for ZenithOS

## Current Status
- Early development
- Booting & kernel experiments

> A Vietnamese open-source OS journey.

> This project is built for learning and research purposes.
